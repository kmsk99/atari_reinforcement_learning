### 선행 조건

이 프로젝트를 실행하기 위해 다음이 필요합니다:

- Python 3.6 이상
- pytorch-cuda=11.8 이상
- anaconda3 이상

### 설치

1. 이 프로젝트의 저장소를 복제합니다:

   ```
   git clone [저장소 URL]
   ```

2. 필요한 라이브러리를 설치합니다:

   ```
   conda env create -f environment.yml

   ```
